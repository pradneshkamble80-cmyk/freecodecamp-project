- # Sea Level Predictor

This project analyzes historical sea level data and predicts future sea level rise up to the year 2050 using linear regression.

## Overview

The dataset contains global average sea level changes from 1880 to 2014. Using this data, the project visualizes trends and builds predictive models to estimate future sea levels.

## Technologies Used

* Python
* Pandas
* Matplotlib
* SciPy

## Features

* Scatter plot of sea level vs year
* Line of best fit using all available data (1880–2014)
* Second line of best fit using recent data (from 2000 onward)
* Future prediction extended to the year 2050

## How It Works

1. Load dataset using Pandas
2. Plot historical sea level data
3. Apply linear regression on:

   * Full dataset
   * Data from year 2000 onward
4. Extend both regression lines to predict sea level rise through 2050

## Output

The program generates a visualization with:

* Blue scatter points (historical data)
* Red line (overall trend)
* Green line (recent trend)

## File Structure

* `sea_level_predictor.py` → main analysis code
* `epa-sea-level.csv` → dataset
* `main.py` → testing script
